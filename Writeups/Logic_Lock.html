🧪 Challenge – Logic Lock (Misc)

It turned out suspect’s apartment has an electronic lock. After analyzing the PCB and looking up the chips you come to the conclusion that it’s just a set of logic gates!

🧠 Solution Process

Having learnt logic gates in my circuits course at University, this puzzle was fairly simple to solve. We are given the end output, so to determine which inputs are set, we just need to work backwards using the various logic gate functions.
